# Teachable Machine - Waste Classification

## Project Description
This project uses **Teachable Machine** to classify different types of waste based on images.  
The goal is to help with everyday waste sorting and promote environmental awareness.

The model classifies images into four categories:
- **Paper**  
- **Plastic**  
- **Glass**  
- **Bio**

---

## How It Works
1. A dataset of images for each category was collected (at least 20–30 images per class).  
2. The images were uploaded to Teachable Machine to train the model.  
3. The trained model can predict the category of a new image.

---

## Usage
1. Open [Teachable Machine](https://teachablemachine.withgoogle.com/)  
2. Upload your trained model or the provided `.zip` file  
3. Test the model using new images

---

## Example Images
- **Paper:** newspapers, notebooks, cardboard  
- **Plastic:** PET bottles, plastic containers  
- **Glass:** glass bottles, jars  
- **Bio:** fruit peels, food scraps, organic waste

---

## Conclusion
This project demonstrates how **AI can assist with simple, everyday tasks**.  
The model can help sort waste at home or school, supporting recycling and environmental protection.
